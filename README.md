# Women Safety App 📱🛡️

An Android-based Women Safety Application designed to provide quick emergency assistance during unsafe situations. The app allows users to store emergency contacts and send alerts through simple actions such as shake detection, helping improve personal safety and emergency response.

## 🚀 Features

* 📞 Store and manage emergency contacts
* 📳 Shake detection for instant emergency activation
* 🚨 Send emergency alerts to saved contacts
* 📍 Share location during emergencies
* 🔄 Background safety monitoring service
* 💾 Local database for contact management
* 🎯 Simple and user-friendly interface

## 🛠️ Tech Stack

* Java
* Android Studio
* SQLite Database
* Android Sensors API
* Android Services
* XML Layouts

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/AkashRthite04/Women-Safety-App.git
```

2. Open the project in Android Studio.

3. Sync Gradle dependencies.

4. Connect an Android device or start an emulator.

5. Run the application.

## 📋 Requirements

* Android Studio Hedgehog or later
* Android SDK
* Java 8+
